Ember Layout
==============================================================================

All you need to implement dynamic layout in Ember.js


Compatibility
------------------------------------------------------------------------------

* Ember.js v3.12 or above
* Ember CLI v2.13 or above
* Node.js v10 or above


Installation
------------------------------------------------------------------------------

```
ember install ember-layout
```
